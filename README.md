# Immigration Solution & Services

A professional, bilingual (English/Arabic) single-page landing page for immigration services.

## Features

- **Bilingual Support**: Toggle between English and Arabic with full RTL support
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern Tech Stack**: Built with React, Tailwind CSS, and Lucide icons
- **Professional Design**: Clean, trustworthy aesthetic inspired by leading immigration websites
- **Service Categories**: Four organized service sections for easy navigation

## Tech Stack

- React 18 (Functional Components with Hooks)
- Vite (Build tool)
- Tailwind CSS (Styling)
- Lucide React (Icons)
- Google Fonts (Merriweather & Inter)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/moabdel/immigrationweb.git
cd immigrationweb
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```

## Sections

1. **Navigation Bar**: Logo, menu links, and language toggle
2. **Hero Section**: Compelling headline with call-to-action
3. **Services**: Four categorized service cards
   - Visas & Family
   - Status & Citizenship
   - Legal Documents
   - Professional Services
4. **Why Choose Us**: Three key value propositions
5. **Footer**: Contact information and social media links

## Customization

- Update contact information in `src/App.jsx` (footer section)
- Modify color scheme in `tailwind.config.js`
- Replace hero background image URL in the hero section
- Add your logo by replacing the text logo in the navbar
